# たべものたんけんたい 🍎🥦

**目的**: 5歳児が楽しく食を学び、日々の食事・早口ことばの発音を通して口腔機能の向上を図るWebアプリ

---

## ✅ 完成済み機能

### 認証・ログイン
- ログイン画面でのユーザー種別選択（こども / 先生管理者）
- 初回登録画面（子ども名）

### こども画面 (child.html)
- 今日のスタンプカード（最大5個）
- 🥦 たべもの学習（9種類のたべもの、詳細表示・学習済みマーク）
- 🎤 はやくちことば練習（8フレーズ、進捗バー）
- ❓ クイズ（6問、正誤判定、スコア記録）
- 🏅 バッジシステム（達成条件付き）
- 動画視聴画面(5歳児向けの食育動画)
- 咀嚼ガム記録画面(数字入力画面)
- 活動ログのAPI保存


### 先生・管理者画面 (teacher.html)
- クラス全体のサマリー（こど数・平均スコア・最高スコア）
- 子ども一覧（名前検索・クラスフィルター・ソート）
- 子ども詳細モーダル（進捗・ログ・コメント記入）
- Chart.js によるクラス全体スコアバー
- 全ユーザー一覧（ロール別フィルター・名前/メール検索）
- ユーザー削除機能（確認ダイアログ付き）
- 活動ログ全体閲覧（種別・名前フィルター）
- 統計チャート（ユーザー構成パイ・活動スコア・こど別スコア）
- 動画投稿画面(動画のPCファイルからのアップロード)
- 咀嚼ガム結果閲覧画面(全体結果まとめ棒グラフ)

## 🗄️ データモデル

### users テーブル
| フィールド | 型 | 説明 |
|-----------|------|------|
| id | text | UUID |
| child_name | text | こどもの名前 |
| role | text | child/parent/teacher/admin |
| avatar | text | アバター番号(1-10) |
| class_name | text | クラス名 |

### activity_logs テーブル
| フィールド | 型 | 説明 |
|-----------|------|------|
| id | text | UUID |
| user_id | text | こどものID |
| child_name | text | こどもの名前 |
| activity_type | text | food_study/pronunciation/quiz |
| score | number | スコア(0-100) |
| completed | bool | 完了フラグ |
| details | text | 活動詳細 |
